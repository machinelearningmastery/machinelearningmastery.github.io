# Quick list of interesting topics based off recent interviews:

Linear Regression / Logistic Regression
Decision Tree / Random Forests
Ensemble Methods
Regularization (L2, Entropy, etc.)
Bias / Variance Tradeoff
Dimensionality Reduction (such as PCA)
Data augmentation and "scale-invariant" features
Optimization including Gradient Descent
Clustering (KMeans, KNearestNeighbors, Approximate KNN w. Annoy)
Representation (for NLP this would sort of be Word Embeddings vs. TFIDF vectors)
Kernel Methods
How to preprocess data
How to deploy a ML model in production
Multilayer Perceptrons (aka Neural Networks)
Feel free to use/ignore any!

My other recommendation for getting started is to try to implement / run examples of what you are reading about. There are a few tools that I think are particularly useful for this (the ones I’ve listed are all in Python, but I imagine there are similar tools in other languages).

1. Numpy - Fast N-Dimensional Arrays (read more)
2. Scikitlearn - Implements many well known machine learning algorithms
3. Pytorch - Very useful for building models that use gradient descent (such as Neural Networks).
4. Jupyter - A nice way to experiment with code in an “interactive notebook”.
5. Matplotlib (and Pandas) - A nice way to visualize results from experiments.
6. Bonus: Visdom - Similar to matplotlib, but is particular useful for live data visualization (as opposed to running visualization after your experiments have complete).
